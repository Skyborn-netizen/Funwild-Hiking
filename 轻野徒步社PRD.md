# 轻野徒步社 - 产品需求文档 (PRD)

## 产品概述

### 产品定位
轻野徒步社是一款专为户外徒步爱好者打造的移动端社群应用，旨在为用户提供活动组织、报名管理、成员交流等一站式服务。

### 目标用户
- 户外徒步爱好者
- 户外运动组织者
- 户外活动参与者

### 产品特色
- 简洁直观的移动端界面设计
- 卡片化布局，层次分明
- 微信生态集成，便于社交分享
- 完整的活动生命周期管理

## 页面结构与功能

### 1. 登录页 (Login Page)

#### 页面布局
- 全屏居中布局
- Logo展示区
- 产品标题与副标题
- 登录方式选择
- 用户协议链接

#### 功能特性
| 功能 | 描述 | 触发方式 |
|------|------|----------|
| 微信一键登录 | 模拟微信授权登录 | 点击"微信一键登录"按钮 |
| 手机号登录 | 通过手机号+验证码登录 | 点击"手机号登录"按钮 |
| 用户协议 | 查看服务条款和隐私政策 | 点击协议链接 |

#### 交互说明
- 登录成功后自动跳转到社团主页
- 手机号登录需填写手机号和验证码
- 用户需同意协议才能完成登录

---

### 2. 社团主页 (Community Page)

#### 页面布局
- 顶部标题栏
- 社团公告区
- Tab导航栏
- 内容展示区
- 发布活动按钮（管理员/领队可见）
- 底部导航栏

#### 功能模块

##### 2.1 社团公告
- **显示内容**: 最新公告信息
- **样式**: 突出显示，背景色区分

##### 2.2 Tab导航
- **活动Tab**: 显示近期活动列表
- **成员Tab**: 显示社团成员列表
- **交互**: 点击切换内容，卡片化按钮设计

##### 2.3 活动列表
- **布局**: 卡片化设计，每个活动一张卡片
- **卡片信息**:
  - 活动标题
  - 时间、地点、强度等级
  - 报名进度
  - 状态标签
  - 操作按钮

##### 2.4 成员列表
- **布局**: 卡片化设计，每个成员一张卡片
- **成员信息**:
  - 头像/昵称
  - 角色标识（管理员/领队/成员）
  - 卡片化展示

#### 数据展示

**活动卡片包含字段**:
```javascript
{
    id: 活动ID,
    title: "活动标题",
    date: "2024-03-15",
    time: "04:30",
    location: "活动地点",
    strength: 4, // 1-5星强度
    maxParticipants: 20, // 最大参与人数
    currentParticipants: 12, // 当前报名人数
    status: "available", // available/full/ended
    description: "活动描述",
    equipment: "装备要求",
    cost: "费用说明",
    organizer: {
        name: "组织者姓名",
        avatar: "头像标识"
    },
    registeredMembers: [
        { name: "成员姓名", avatar: "头像标识" }
    ]
}
```

**成员卡片包含字段**:
```javascript
{
    name: "成员姓名",
    role: "admin/leader/member", // 角色
    avatar: "头像标识"
}
```

## 弹窗组件

### 1. 报名确认弹窗 (Join Modal)

#### 弹窗内容
- 活动信息展示
- 支付说明
- 微信收款二维码
- 操作按钮

#### 交互流程
1. 点击"立即报名"触发
2. 展示活动信息和支付说明
3. 用户确认支付后点击"我已完成支付"
4. 更新活动报名状态
5. 关闭弹窗并提示成功

### 2. 手机号登录弹窗 (Phone Login Modal)

#### 弹窗内容
- 手机号输入框
- 验证码输入框
- 获取验证码按钮
- 登录/取消按钮

#### 交互流程
1. 点击"手机号登录"触发
2. 填写手机号和验证码
3. 点击登录完成认证
4. 关闭弹窗并跳转主页

## 用户角色与权限

### 角色定义

#### 管理员 (admin)
- 查看完整成员名单
- 发布活动权限
- 管理社团事务
- 查看所有活动数据

#### 领队 (leader)
- 发布活动权限
- 查看活动报名详情
- 协助活动组织

#### 成员 (member)
- 浏览活动信息
- 报名参与活动
- 查看个人信息
- 查看已报名活动

### 权限矩阵

| 功能 | 管理员 | 领队 | 成员 |
|------|--------|------|------|
| 发布活动 | ✓ | ✓ | ✗ |
| 查看完整名单 | ✓ | ✓ | ✗ |
| 报名活动 | ✓ | ✓ | ✓ |
| 浏览活动 | ✓ | ✓ | ✓ |
| 个人中心 | ✓ | ✓ | ✓ |

## 交互设计细节

### 页面切换动画
- **效果**: 滑动切换 + 淡入淡出
- **时长**: 300ms
- **缓动**: cubic-bezier(0.4, 0, 0.2, 1)

### 按钮交互状态
- **默认状态**: 灰色文字 + 白色背景
- **激活状态**: 白色文字 + 深绿色背景
- **悬停效果**: 轻微阴影变化
- **点击反馈**: 按钮按下效果

### 表单交互
- 输入框聚焦效果
- 验证提示信息
- 按钮禁用状态
- 加载中状态

### 卡片设计
- **阴影**: 轻微边框，无阴影
- **间距**: 统一的内外边距
- **圆角**: 统一的圆角半径
- **背景**: 白色主体，浅绿容器

## 业务流程

### 用户注册登录流程
1. 进入应用 → 登录页
2. 选择登录方式（微信/手机号）
3. 完成认证 → 跳转主页
4. 显示底部导航栏

### 活动参与流程
1. 浏览活动列表
2. 查看活动详情
3. 点击"立即报名"
4. 确认支付信息
5. 完成支付确认
6. 更新报名状态
7. 成功提示

### 活动发布流程（管理员/领队）
1. 点击"发布活动"按钮
2. 填写活动信息（后续版本功能）
3. 发布活动到列表
4. 其他用户可报名

### 页面导航流程
1. 底部导航切换页面
2. Tab导航切换内容
3. 页面状态保持
4. 数据动态加载
